

This project analyzes the Annual Enterprise Survey 2023 (New Zealand) dataset to understand financial performance across industries.
The dataset includes information on income, expenditure, and profitability of various industries.

Tools & Libraries Used
Library
Purpose
Pandas
Data cleaning & analysis
NumPy
Numerical operations
Matplotlib
Data visualization
Seaborn
Advanced charting & styling

Key KPIs
■ Total Revenue: 11,819,593
■ Total Expense: 10,487,640
■ Total Profit: 1,331,953
■ Profit Margin: 11.27%

Insights
• Revenue shows a steady upward trend from early years to 2023.
• Profit margin fluctuates slightly, indicating changes in cost efficiency.
• Service sectors like Retail, Manufacturing, and Construction dominate total revenue.
• Operational efficiency improved in 2023 compared to 2022.

Recommendations
1. Focus investment in high-performing industries (Retail, Manufacturing).
2. Monitor expenditure trends for cost optimization.
3. Use predictive analytics to forecast 2024 revenue & profit.
4. Integrate this analysis into Power BI / Tableau Dashboard for real-time monitoring.
6. Author Information
Role: Data Analyst / Machine Learning Enthusiast Tools Expertise: Python, Power BI, Tableau, SQL
